HTML Documents


The set of tags necessary for any HTML web page you create is known as the basic HTML skeleton. Without the skeleton, HTML files will not be rendered properly in web browsers because the tags that make up the skeleton inform browsers of the type of file it is viewing.

!DOCTYPE html

There is no HTML tag in the declaration. The browser receives "information" from it about the type of document to anticipate.

< html > < /html >

< head > < /head >

All other HTML elements—aside from the!DOCTYPE> tag—are contained within it. Additionally, it stands for the HTML document's root

< title>< /title >

It defines the document's title. The page's tab or the title bar of the browser both display the title, which must only be text. For search engine optimization (SEO), the content of a page title is crucial! When listing pages in search results, search engine algorithms use the page title to determine the hierarchy.

< body >< /body>

There are headings, paragraphs, images, hyperlinks, tables, lists, and other elements that make up an HTML document's body.




HTML Headings


The text headers for a document are made using the h1, h2, h3, h4, h5, and h6 tags. They can choose from six different font sizes to show the header text. The font size gradually decreases from the largest size, h1, to the smallest size, h6, as indicated by the integer number.



HTML Paragraph

An HTML paragraph is represented by the < p > element.


HTML Bold Text

Bold text is indicated via the < b > element without any further significance.



HTML Italic Text

The italicized text is styled with the < i > element to set it apart from the surrounding text without giving the words any additional prominence.



HTML Small/Big Text

- The HTML < small > tag reduces the font size of the text in the document by one.

- The HTML < big > tag enlarges text in the document by one font size


HTML Comment Text

A comment is text enclosed within in HTML. < !╌ ╌ > tags. By using this technique, you can inform a browser that some elements are comments and shouldn't be displayed on the front end.



HTML Mark Text

In HTML, the marked text is specified via the < mark > tag. It is used to draw attention to a specific section of text inside a paragraph.



HTML Delete Text

The HTML < del > element, which stands for "delete," is used to indicate text that has been removed from the document.



HTML Emphasized Text

The < em > tag is used to define emphasized text. The content inside is typically displayed in italic.



HTML Inserted Text

When text is entered into a document, the < ins > tag describes it. Usually, inserted text is underlined by browsers.



HTML Subscript Text

Subscript text is described by the tag < sub > Subscript text is often displayed in a smaller font and sits half a character below the main line.



HTML Superscript Text

The HTML element < sup > designates inline text that should be shown as superscript exclusively for typographical reasons. Superscripts are often displayed using a higher baseline and smaller font.



HTML Line Break

The line break should be indicate by the < br > tag.



HTML Horizontal

To make a horizontal line use the HMTL < hr > tag.


HTML Quotation

A block of text that is quoted from another source is designated using the HTML < blockquote > tag. The blockquote> tag's content is typically displayed by browsers as indented text.



HTML Short Quotation

The < q > tag is a standard quotation tag and used for short quotation.



HTML Contact Information

To get the website owner's or article author's contact information, use the < address > tag. Address, phone number, email, website connection, and a lot more could be included.



HTML Work Title

The source of a quotation or creative work is indicated with the < cite > element. Instead of the author or creator of a referenced creative work, use the element to identify the name.



HTML List Items

An unordered list of elements is represented by the HTML element < ul >, which is commonly shown as a bulleted list.



A list item that is a part of an ordered and unordered list of items is defined by the < li > element.


An ordered list is defined by the < ol > tag. A list that is arranged can be alphabetical or numerical.



HTML Description Items

A description list is defined by the tag < dl >


In a description list, the < dt > tag defines a term or name.


An item in a description list can be described using the < dd > tag. You can insert lists, paragraphs, line breaks, photos, links, and more inside a < dd > tag.



HTML Table Items

The HTML element < table > represents tabular data, or data shown in a two-dimensional table with rows and columns of data-containing cells.


The HTML element < th > designates a cell as the group's header. The scope and headers attributes specify exactly what this group is.


To indicate a table row within a table, use the HTML < tr > tag.


A typical data cell in an HTML table is defined using the < td > tag.



HTML Caption Table

The title of a table is specified in the HTML text by the < caption > tag.



HTML One or More Table

The < colgroup > element designates a group of one or more columns in a table for formatting.



HTML Images

- The < img > tag is used to embed an image in an HTML page.

- src Specifies the path to the image

- alt Specifies an alternate text for the image, if the image for some reason cannot be displayed



HTML Videos

- A movie clip or other video streams can be included in a document by using the < video > tag.

- Loop Specifies that after each complete cycle, the video will begin again.

- Controls Indicates that video controls, such as a play/pause button, should be visible.

- Before the user clicks the play button or while the movie is downloading, the Poster specifies an image to be displayed.


HTML YouTube Videos

- An HTML page can be embedded inside of another one using the < iframe > HTML element, which symbolizes nested browsing contexts

- The document's address is specified by the src attribute in an < iframe >

- Allowfullscreen is an HTML attribute that is frequently applied to the iframe HTML tag. It allows fullscreen video playback from a variety of sources.

- To embed a video in an HTML page, use the < iframe > element. The source attribute included the video URL. For the dimensions of the video player, set the width and height of the video appropriately.


HTML Links

- With the help of the < href > property on the < a > HTML element, you may link to any URL-enabled object, including files, web pages, email addresses, places on the same page, and more

- Each < a > tag should contain information that identifies the location of the link. Pressing enter while keeping your attention on the < a > element will activate the < href > attribute if it is there

- To open the linked document, use the < target > attribute to specify a location

- The connected page is opened in a new window or tab using the code < _blank >



HTML Navigation Links

A section of a page that serves as a source of navigation links, either to other documents or to other pages on the same website, is designated with the HTML element < nav >. Indexes, menus, and tables of contents are typical illustrations of navigation sections.



HTML Main Content of a Document

The content of a webpage that relates to the main subject of that page or application is designated using the < main > element. It should only contain information that is specific to that page and not information that is repeated on other webpages, such as headers, footers, and the main navigation.



HTML Header of a Page

The header of a page or section is specified by the < header > tag. Typically, it has a logo, a search bar, navigational links, etc.



HTML Footer for Documents or Section

The footer for its closest ancestor sectioning content or sectioning root element is represented by the HTML element < footer >. Typically, a footer includes information about the section's author, copyright information, or links to related papers.



HTML Article

The < article > HTML element represents a self-contained composition in a document, page, application, or site that is meant to be independently distributable or reusable. Examples include a forum post, a magazine, newspaper, or blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent piece of content.



HTML Section in Document

An HTML structural element called < section > is used to gather together items that are linked to one another. Each < section > normally consists of a heading element or elements that convey relevant material.



HTML Aside from the Content

The HTML element known as < aside > designates a section of a page whose content is only loosely related to the document's primary text. Frequently, sidebars or call-out boxes are used to present asides.
