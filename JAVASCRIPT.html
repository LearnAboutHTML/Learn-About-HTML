What is JAVASCRIPT

-Javascript (JS) is a scripting language that is mostly utilized on the Internet. It is typically seen integrated in HTML code and is used to improve HTML pages. JavaScript is an interpreted programming language. As a result, it does not need to be compiled. JavaScript is used to create interactive and dynamic web sites.


-Javascript is a programming language used by programmers all over the globe to develop dynamic and interactive online content such as programs and browsers. JavaScript is the most widely used programming language in the world, with 97.0 percent of all websites using it as a client-side programming language.

Client-Side JAVASCRIPT

There are various JavaScript methods available on the client side:

*To write in a document, use document.write().
*To open a pop-up window, use alert().
*prompt() is used to solicit user input
*getElementById() returns the element with the given ID.
*property with the given value


Inserting JAVASCRIPT

There are three typical methods for putting JavaScript code into a page. web site

*Inside an HTML tag script
*In an external file
*As a value of some HTML attributes
